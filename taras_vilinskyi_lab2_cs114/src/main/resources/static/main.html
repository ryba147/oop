Для зручності тестування було мінімально оформлено фронтенд :)
  • Щоб користуватись сайтом, необхідно зареєструватись.
Ім'я та пароль повинні бути довжиною >= 3 символів, інакше акаунт не буде створено.
При покупці квитка одним з користувачів, квиток уже не буде доступним для покупки іншим користувачам.
Коли квитки закінчуються, відбувається генерація п'яти нових.
Після продажу квитка, він повертається до каси.
Щоб провести пошук за назвою(BUS_/PLANE_/TRAIN_TICKET), спершу необхідно переглянути усі доступні квитки.

  • Купуйте (id квитка можна переглянути обираючи [View available tickets])

  • Переглядайте доступні квитки
  • Переглядайте придбані квитки
  • Проводьте пошук за типом квитка
